# Word文件重複內容檢測工具

[English](README.md) | [简体中文](README.zh_Hans.md) | [繁體中文](README.zh_Hant.md)

一個用於檢測 Word 文件中重複內容並新增智慧註解的開源工具, 具有智慧分組和精確段落定位等功能.

## ✨ 功能特點

- **🔍 自動檢測**: 智慧檢測 Word 文件中的重複內容;
- **📝 智慧註解**: 為重複內容新增註解, 顯示重複來源段落號;
- **🔢 智慧分組**: 相同內容使用相同編號, 便於識別;
- **🎯 精確定位**: 精確段落定位, 無頁碼估算誤差;
- **⚙️ 可自訂參數**: 支援自訂檢測參數 (預設30字元);
- **📄 非破壞性**: 產生帶註解的新文件, 原文件不變;
- **💻 免安裝**: Portable 運行, 無需安裝, 沒有額外依賴;
- **🌐 多語言支援**: 支援英語、簡體中文、繁體中文;

## 🎮 使用方法

直接從 [Releases](https://github.com/fernvenue/word-duplicate-detector/releases/latest) 下載最新版本運行即可, 無需安裝.

![image](./assets/images/3B96705F8401E35A.webp)

1. **選擇文件**: 選擇要檢測的 Word 文件 (.docx格式);
2. **設定參數**: 設定最小重複長度 (建議20-50字元);
3. **開始檢測**: 點擊 *開始檢測* 按鈕;
4. **檢視結果**: 檢查產生的帶註解文件;
5. **瀏覽重複**: 透過重複編號快速識別重複組;

### 📊 註解格式

工具以以下格式新增註解:

```
重複#1: 與第25段內容重複
```

- **重複#N**: 同一內容的所有重複使用相同編號;
- **第Y段**: 重複內容來源的精確段落編號;

### 🧠 重複編號邏輯

相同的重複內容會被分配相同的編號, 例如: A 段落重複 B 段落, C 段落也重複 B 段落, 則 A 和 C 都會顯示相同的重複編號, 便於快速識別哪些段落是同一組重複.

## 🚀 自行構建

需要預先安裝好 Python 環境.

克隆倉庫:

```bash
git clone https://github.com/fernvenue/word-duplicate-detector.git
cd word-duplicate-detector
```

運行構建腳本:

```bash
build.bat
```
